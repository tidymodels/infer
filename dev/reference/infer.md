# infer: a grammar for statistical inference

The objective of this package is to perform statistical inference using
a grammar that illustrates the underlying concepts and a format that
coheres with the tidyverse.

## Details

For an overview of how to use the core functionality, see
[`vignette("infer")`](https://infer.tidymodels.org/dev/articles/infer.md)
